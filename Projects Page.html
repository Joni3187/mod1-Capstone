PacMen Exercise

This project was a continuation from a previous assignment, where we were able to manipulate the movements of PacMan across the browser window. In this exercise we created a web page that produces a PacMan on the click of a button by using the array methods we learned previously and new techniques for manipulating the DOM. This has been one of my favorite projects! PacMan is such an iconic character, getting to incorporate him into our code was great application!

Eye Exercise

The Eye Exercise was a fun project utilizing our new skills and showing what we can do when we combine HTML, CSS, and JavaScript. We created a web page with two eyes that follow your mouse movement. What seemed to me like a complicated project turned out to be a surprising small amount of code!
